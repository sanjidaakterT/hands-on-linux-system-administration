# Hands-On Linux System Administration & Maintenance

This repository contains the practical lab presentation and documentation for basic Linux system administration tasks, designed to demonstrate core operating system concepts and user management.

## Project Overview
* **Topic:** System Information Exploration & Security Maintenance
* **Platform:** Ubuntu Server / Linux Environment
* **Design & Documentation:** Created as a professional technical flyer using Canva

## Key Tasks Documented:
1. **Viewing System Information:** Checking OS details, RAM capacity, and processor architecture.
2. **User Security:** Step-by-step documentation for user password configuration and modification.
